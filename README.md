# Next.js Vercel Deployment

## Description
This is a simple Next.js application deployed on Vercel. It showcases a few API endpoints and a basic frontend.

## What's Built
- A Next.js app with TypeScript
- Two API endpoints: `/api/hello` and `/api/goodbye`
- Deployed on Vercel

## How to Run
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
4. Open `http://localhost:3000` in your browser.

## API Documentation
- **GET /api/hello**: Returns a greeting message.
- **GET /api/goodbye**: Returns a farewell message.

## Environment Variables
- `NEXT_PUBLIC_API_URL`: Base URL for the API (default: `/api`).
